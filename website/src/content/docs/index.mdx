Pompelmi adds safe, fast malware scanning to your file uploads.

- **UI:** ready-made upload button & dropzone for React/Next.js.  
- **Server:** Express/Koa middlewares, Next.js API route helper.  
- **Safety:** size & MIME guards, optional YARA rules, clear verdicts.

**Try the live demo** on the homepage: drag a clean JPG → **CLEAN**; try EICAR → **MALICIOUS**.
